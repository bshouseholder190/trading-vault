# GitHub Setup

Run these commands once after creating the repo on GitHub:

```powershell
cd "C:\Users\bshou\OneDrive\Documents\Claude\Projects\Trading Vault"
git remote add origin https://github.com/bshou/trading-vault.git
git branch -M main
git push -u origin main
```

Replace `bshou/trading-vault` with your actual GitHub username and desired repo name.
